#  Game of Set

An implementation of the [Set game](https://en.wikipedia.org/wiki/Set_(card_game)) written in SwiftUI for assignment 3 of [Stanford University's course CS193p](https://cs193p.sites.stanford.edu) (Developing Applications for iOS using SwiftUI) of Spring 2020.

## Introduction

In this card game, each card has 4 kinds of attributes: 

- Color
- Shape
- Shading
- Number of Shapes

The goal is to select 3 cards which have for each kind of attribute:

- either 3 times the same attribute
- or 3 times the same attribute
